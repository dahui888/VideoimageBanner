🔥 图片+视频混合banner
  ## gradle接入
	
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	
	dependencies {
		
	        implementation 'com.github.jkdsking:VideoimageBanner:1.0.1'
	
	}
  
## 具体使用请查看demo (图片加载使用glide 视频播放使用饺子播放器)
